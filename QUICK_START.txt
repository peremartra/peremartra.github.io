QUICK START - 3 STEPS
=====================

STEP 1: ADD PHOTO (2 mins)
---------------------------
→ Save your photo as: assets/images/profile.jpg
→ Delete the ADD_PHOTO_HERE.txt file

STEP 2: UPDATE INFO (2 mins)
-----------------------------
→ Edit _config.yml:
   - Change email
   - Change linkedin_username
   - Change twitter_username (or remove if you don't have)

STEP 3: UPLOAD TO GITHUB (1 min)
---------------------------------
→ Create repo: peremartra.github.io
→ Drag all files to the repo
→ Settings → Pages → Enable (branch: main)
→ Wait 2 minutes
→ Visit: https://peremartra.github.io

DONE! ✅
